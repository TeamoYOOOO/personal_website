WorkerBee
Java, MySQL
A project to automatically find related jobs for users based on the customized background, which effectively reduce the unnecessary work for people to find jobs from a huge number of alternatives from the internet.
  • Utilized JavaServer Pages to develop the project, helping users to find related jobs based on the customized background
  • Made use of the Glassdoor dataset from world’s largest data science community Kaggle as the data of the project
  • Deployed Google OAuth as the authentication and authorization module to ensure the security and the reliability
  • Adopted MySQL as the database to store the information of the job details, background of the user, etc. to make customized recommendation
  • Followed DAO pattern in Object-Oriented-Design to integrate different components and increase code reusage
